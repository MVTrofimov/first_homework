Программа запускается в классе Main в нем происходит инициализация контекста приложения. В зависимости от того какой активный профиль указан в файле applicaton.properties, будут созланы разные бины для взаимодействия с репозиторием оба он реализуют интерфейс Repository. Отличие класса InitRepository от DefaultRepository заключается только в наличии метода для инициализации хранилища из пользовательского файла. В зависимости от профиля буду использовать разные файлы со свойствами приложения. При использовании профиля init приложение будет вначале инициализировать хранилище из пользовательского файла, а после добавления или удаления сохранять результат в этот же файл. При использовании профиля default приложение будет добавлять и удалять информацию о контактах и также может сохранить результат в файл путь, к которому находится в соотвествующем файле со свойствами (application_default.properties).
